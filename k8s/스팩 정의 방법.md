# 파드 스팩 정의
```yaml
# pod-example.yaml
apiVersion: v1
kind: Pod
metadata:
  name: my-pod
  labels:
    app: my-app
spec:
  containers:
    - name: my-container
      image: nginx
      ports:
        - containerPort: 8080

```


# 서비스 스팩 정의
```yaml
# service-clusterip.yaml
apiVersion: v1
kind: Service
metadata:
  name: my-service
spec:
  selector: # 라벨을 통해 연결할 파드를 등록
    app: my-app
  ports:
    - protocol: TCP
      port: 80       # 클라이언트에서 접근하는 서비스 포트
      targetPort: 8080  # 파드에서 노출된 포트

```

```bash
kubectl apply -f service-clusterip.yaml
```
